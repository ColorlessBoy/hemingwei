1. Random Controlled Trials#

1.1. Prepare#

问题一：我们从两个正态分布中采样 \(x_1 \sim \mathcal{N}(\mu_1, \sigma^2_1)\) 和 \(x_2 \sim \mathcal{N}(\mu_2, \sigma^2_2)\)，那么 \(x_1 + x_2\) 服从什么分布呢？

\begin{aligned} P(x_1 + x_2 \le t) =& \int^{\infty}{-\infty} p_1(x_1) \int^{t - x_1}{-\infty} p_2(x_2) \mathrm{d} x_2 \mathrm{d} x_1 \ =& \int^{\infty}_{-\infty} p_1(x_1) P_2(t - x_1) \mathrm{d} x_1 \ \end{aligned}

\begin{aligned} p(t) =& \int^{\infty}{-\infty} p_1(x_1) p_2(t - x_1) \mathrm{d} x_1 \ =& \int^{\infty}{-\infty} \frac{1}{\sqrt {2 \pi} \sigma_1}\exp\bigg(-\frac{(x_1 - \mu_1)^2}{2\sigma^2_1}\bigg) \frac{1}{\sqrt {2 \pi} \sigma_2}\exp\bigg(-\frac{(t - x_1 - \mu_2)^2}{2\sigma^2_2}\bigg) \mathrm{d} x_1 \ =& \int^{\infty}{-\infty} \frac{1}{{2 \pi} \sigma_1 \sigma_2}\exp\bigg( - \frac{\sigma^2_1 + \sigma^2_2}{2\sigma^2_1 \sigma^2_2} x_1^2 + \frac{\sigma^2_2 \mu_1 + \sigma^2_1 (t - \mu_2)}{\sigma^2_1 \sigma^2_2} x_1 - \frac{\mu^2_1 \sigma^2_2 + (t - \mu_2)^2 \sigma^2_1}{2\sigma^2_1 \sigma^2_2} \bigg)\mathrm{d} x_1 \ =& \int^{\infty}{-\infty} \frac{1}{{2 \pi} \sigma_1 \sigma_2}\exp\bigg[

  • \frac{\sigma^2_1 + \sigma^2_2}{2\sigma^2_1 \sigma^2_2} \bigg( x_1 - \frac{\sigma^2_2 \mu_1 + \sigma^2_1(t - \mu_2)^2}{\sigma^2_1 + \sigma^2_2} \bigg)^2

  • \frac{(t - \mu_1 - \mu_2)^2}{2(\sigma^2_1 + \sigma^2_2)} \bigg] \mathrm{d} x_1 \ =& \exp \bigg( - \frac{(t - \mu_1 - \mu_2)^2}{2(\sigma^2_1 + \sigma^2_2)} \bigg) \cdot \frac{1}{{2 \pi} \sigma_1 \sigma_2} \int^{\infty}_{-\infty} \exp\bigg[

  • \frac{\sigma^2_1 + \sigma^2_2}{2\sigma^2_1 \sigma^2_2} \bigg( x_1 - \frac{\sigma^2_2 \mu_1 + \sigma^2_1(t - \mu_2)^2}{\sigma^2_1 + \sigma^2_2} \bigg)^2 \bigg] \mathrm{d} x_1 \ =& \exp \bigg( - \frac{(t - \mu_1 - \mu_2)^2}{2(\sigma^2_1 + \sigma^2_2)} \bigg) \cdot \frac{1} \ =& \frac{1}{\sqrt{2\pi (\sigma^2_1 + \sigma^2_2)}} \exp \bigg( - \frac{(t - \mu_1 - \mu_2)^2}{2(\sigma^2_1 + \sigma^2_2)} \bigg) \end{aligned}

综上所述，\(t = x_1 + x_2 \sim \mathcal{N}(\mu_1 + \mu_2, \sigma^2_1 + \sigma^2_2)\)

import sympy as sp
x1, mu1, sigma1, t, mu2, sigma2 = sp.symbols('x_1 \mu_1 \sigma_1 t \mu_2 sigma_2')
poly1 = - (x1 - mu1)**2 / (2 * sigma1**2) - (t - x1 - mu2)**2 / (2 * sigma2**2)
poly2 = sp.Poly(poly1, x1)
coeffs = poly2.coeffs()
display(poly2)
display(coeffs[0])
display(coeffs[1])
display(coeffs[2])

1.2. 从一个简单的例子出发#

三个随机变量满足：

  • \(x \sim \mathcal{N}(\mu_x, \sigma^2_x)\)

  • \(w \sim B(0.5)\), \(p(w = 0) = 0.5\), \(p(w = 1) = 0.5\)

  • \(y(x, w) \sim \mathcal{N}(a_w x + b_w, \sigma^2_w) \)

那么，三个随机变量联合分布满足： \begin{aligned} p(x, w, y) =& \frac{1}{\sqrt{2\pi} \sigma_x} \exp \bigg(- \frac{(x - \mu_x)^2}{2\sigma^2_x}\bigg) \cdot \frac{1}{2} \cdot \frac{1}{\sqrt{2\pi} \sigma_w} \exp \bigg(- \frac{(y - a_w x - b_w)^2}{2\sigma^2_w}\bigg) \ =& \frac{1}{4 \pi \sigma_x \sigma_w} \exp \bigg[ -\frac{\sigma_{w}^{2} + \sigma_{x}^{2} a_{w}^{2}}{2 \sigma_{w}^{2} \sigma_{x}^{2}} \bigg( x - \frac{\mu_{x} \sigma_{w}^{2} - \sigma_{x}^{2} a_{w} b_{w} + \sigma_{x}^{2} a_{w} y}{\sigma_{w}^{2} + \sigma_{x}^{2} a_{w}^{2}} \bigg)^2 - \frac{(y - \mu_{x} a_{w} - b_{w})^2}{\sigma_{w}^{2} + \sigma_{x}^{2} a_{w}} \bigg] \end{aligned}

\begin{aligned} p(w, y) =& \int^\infty_{-\infty} p(x, w, y) \mathrm{d} x \ =& \frac{1}{4 \pi \sigma_x \sigma_w} \frac{2 \pi \sigma_{w} \sigma_{x}}{\sqrt{\sigma_{w}^{2} + \sigma_{x}^{2} a_{w}^{2}}} \exp \bigg( - \frac{(y - \mu_{x} a_{w} - b_{w})^2}{\sigma_{w}^{2} + \sigma_{x}^{2} a_{w}} \bigg) \end{aligned}

import sympy as sp
x, mux, sigmax, y, aw, bw, sigmaw = sp.symbols('x \mu_x \sigma_x y a_w b_w \sigma_w')
poly1 = - (x - mux)**2 / (2 * sigmax**2) - (y - aw * x - bw)**2 / (2 * sigmaw**2)
poly2 = sp.Poly(poly1, x)
coeffs = poly2.coeffs()
display(poly2)

display(coeffs[2])
